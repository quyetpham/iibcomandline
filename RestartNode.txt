Các bước restart node
1.topas -m
2.đăng nhập user iibsonth
3.set biến môi trường để chạy lệnh mqsi:
	vào bin trong cài đặt iib : cd app/iib-10.0.0.7/server/bin
	chạy lệnh . ./mqsiprofile
4.Kiểm tra 2 node:
	mqsilist
5.vào toolkit stop hết các dịch vụ.
6.Stop node
	mqsistop IBNODE1
7.theo dõi lại:
	topas -m
	mqsilist (kiểm tra trạng thái xem đã stop hay chưa)
8.Start lại node
	mqsistart IBNODE1
9.Vào toolkit start lại dịch vụ.
10.Kiểm tra lại các service
Done 	

mqsistop -i IBNODE1
